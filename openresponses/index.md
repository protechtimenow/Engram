# Open Responses

Open Responses is an open-source specification and ecosystem for building multi-provider, interoperable LLM interfaces based on the OpenAI Responses API.

Sources:
- [Overview](https://www.openresponses.org/)
- [Specification](https://www.openresponses.org/specification)
- [Reference](https://www.openresponses.org/reference)
- [GitHub](https://github.com/openresponses/openresponses)

## Overview
- **Multi-provider by default**: A shared schema mapping to many model providers.
- **Agentic workflow friendly**: Standardized streaming, tool invocation, and atomic "items".
- **Extensible**: Stable core with room for provider-specific features.

## Getting Started
1. Read the [Specification](https://www.openresponses.org/specification)
2. Review the [Reference](https://www.openresponses.org/reference)
